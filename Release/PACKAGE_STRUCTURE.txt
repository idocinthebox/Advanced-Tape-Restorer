  FIRST-TIME USER QUICK PATH
================================================================================

For new users, follow this exact order:

1. 📖 READ: QUICK_START.txt
   Understand what to expect and basic workflow

2. 🔧 RUN: 1_Install_VapourSynth.bat (as Administrator)
   Wait for installation to complete

3. 🔧 RUN: 2_Install_FFmpeg.bat (as Administrator)
   Wait for installation to complete

4. ✅ RUN: 3_Check_Prerequisites.bat
   Verify all checks pass (should see green checkmarks)

5. 🚀 RUN: Advanced_Tape_Restorer_v3.exe
   Application should launch successfully!

6. 🎬 TEST: Process a short test video
   - Select input file
   - Select output file
   - Use default settings
   - Click "Start Processing"

7. 📖 EXPLORE: Try different features
   - Restoration tab: Process existing videos
   - Capture tab: Capture from tapes (requires hardware)
   - Batch tab: Queue multiple videos

================================================================================
  WHAT EACH FILE DOES
================================================================================

🔧 SETUP SCRIPTS:

1_Install_VapourSynth.bat
  - Downloads VapourSynth R68 from GitHub
  - Runs installer automatically
  - Installs required plugins (havsfunc, ffms2, bm3d)
  - Adds to system PATH
  - Verifies installation

2_Install_FFmpeg.bat
  - Downloads FFmpeg essentials from gyan.dev
  - Extracts to C:\ffmpeg
  - Adds to system PATH
  - Verifies installation

3_Check_Prerequisites.bat
  - Checks if FFmpeg is accessible
  - Checks if FFprobe is accessible
  - Checks if VapourSynth (vspipe) is accessible
  - Shows version information
  - Offers to launch application if all OK

================================================================================
  UNATTENDED INSTALLATION (Advanced)
================================================================================

For IT administrators deploying to multiple PCs:

PowerShell Script Example:
```powershell
# Extract package
Expand-Archive -Path "ATR_v3.0.0.zip" -Destination "C:\Program Files\ATR"

# Install prerequisites silently
Start-Process "C:\Program Files\ATR\1_Install_VapourSynth.bat" -Wait
Start-Process "C:\Program Files\ATR\2_Install_FFmpeg.bat" -Wait

# Create desktop shortcut
$WshShell = New-Object -ComObject WScript.Shell
$Shortcut = $WshShell.CreateShortcut("$env:Public\Desktop\ATR.lnk")
$Shortcut.TargetPath = "C:\Program Files\ATR\Advanced_Tape_Restorer_v3.exe"
$Shortcut.Save()
```

Batch Script Example:
```batch
@echo off
:: Extract package
powershell -command "Expand-Archive -Path 'ATR.zip' -Destination 'C:\ATR'"

:: Install prerequisites
cd C:\ATR
call 1_Install_VapourSynth.bat
call 2_Install_FFmpeg.bat

:: Verify installation
call 3_Check_Prerequisites.bat
```

================================================================================
  PERFORMANCE EXPECTATIONS
================================================================================

PROCESSING SPEED (1 hour VHS tape):

CPU: Intel Core i5 (4 cores)
  - Fast preset: 1-2 hours
  - Medium preset: 2-3 hours
  - Slow preset: 3-4 hours

CPU: Intel Core i7/i9 (8+ cores)
  - Fast preset: 0.5-1 hour
  - Medium preset: 1-2 hours
  - Slow preset: 2-3 hours

Add GPU (NVIDIA):
  - GPU denoising: 20-30% faster
  - AI upscaling: 2-3x slower (but much better quality)

RAM Usage:
  - Typical: 2-4 GB
  - With AI upscaling: 4-8 GB
  - Batch processing: 2-4 GB per job
