v2.0.0 - November 2025
================================================================================

MAJOR FEATURES:
  ✓ Complete architectural redesign with modular structure
  ✓ NEW: Tape capture functionality (analog & DV/miniDV)
  ✓ NEW: DirectShow device detection and management
  ✓ NEW: Capture tab with live device monitoring
  ✓ Enhanced field order auto-detection algorithm
  ✓ Improved preset management system
  ✓ Better error handling and logging

CORE PROCESSING:
  ✓ Modular core engine (independent from GUI)
  ✓ Callback-based progress updates
  ✓ Improved VapourSynth script generation
  ✓ Better source filter auto-selection
  ✓ Enhanced field order detection with configurable thresholds
  ✓ Support for all ProRes variants
  ✓ FFV1 lossless codec support
  ✓ DNxHD professional codec

CAPTURE MODULE (NEW):
  ✓ Analog capture from USB capture cards
  ✓ DV/miniDV capture via FireWire
  ✓ Multiple lossless codecs: HuffYUV, Lagarith, FFV1, UT Video
  ✓ Configurable resolution, framerate, audio settings
  ✓ Auto-process captured video option
  ✓ Device type identification (analog/dv/hdmi)
  ✓ Capture statistics and monitoring

GUI IMPROVEMENTS:
  ✓ Three-tab interface (Restoration/Capture/Batch)
  ✓ Enhanced settings organization
  ✓ Improved batch queue management
  ✓ Better progress visualization
  ✓ Real-time log output with throttling
  ✓ Persistent window state

ARCHITECTURE:
  ✓ Separated core/capture/gui modules
  ✓ API-first design for extensibility
  ✓ Independent module testing
  ✓ Comprehensive test suite
  ✓ Better code organization (~970 lines core vs 2500 monolith)

DOCUMENTATION:
  ✓ Complete API documentation
  ✓ Capture guide with hardware setup
  ✓ Architecture overview for developers
  ✓ Automated setup scripts
  ✓ Comprehensive troubleshooting guide

PERFORMANCE:
  ✓ More efficient subprocess management
  ✓ Better memory handling
  ✓ Improved ETA estimation
  ✓ Optimized progress callbacks

BUG FIXES:
  ✓ Fixed field order detection false positives
  ✓ Improved temp file cleanup
  ✓ Better handling of non-ASCII file paths
  ✓ Fixed audio sync issues in some formats
  ✓ Resolved codec compatibility issues

BREAKING CHANGES:
  - Settings file format changed to hierarchical JSON
  - Some API methods renamed for clarity
  - Requires Python 3.9+ (was 3.7+)
