================================================================================
  ADVANCED TAPE RESTORER v2.0 - QUICK START GUIDE
================================================================================

Welcome! This guide will help you get started in 5 minutes.

================================================================================
  STEP 1: INSTALL REQUIRED DEPENDENCIES
================================================================================

Before running Advanced Tape Restorer, you need two essential tools:

1. VapourSynth (Video Processing Framework)
   - Download: https://github.com/vapoursynth/vapoursynth/releases
   - Install version R68 or newer
   - Choose 64-bit installer
   - Run "1_Install_VapourSynth.bat" for automated setup (see below)

2. FFmpeg (Video Encoding/Analysis)
   - Download: https://ffmpeg.org/download.html#build-windows
   - Extract to C:\ffmpeg (or any location)
   - Run "2_Install_FFmpeg.bat" to add to system PATH automatically

NOTE: Both batch files are included in this package for easy installation!

================================================================================
  STEP 2: VERIFY INSTALLATION
================================================================================

Run "3_Check_Prerequisites.bat" to verify everything is installed correctly.

You should see:
  ✓ FFmpeg found
  ✓ VapourSynth found
  ✓ vspipe found

If any are missing, re-run the appropriate installation script.

================================================================================
  STEP 3: LAUNCH THE APPLICATION
================================================================================

Double-click: Advanced_Tape_Restorer_v2.exe

The application will open with three main tabs:
  - Restoration: Process existing video files
  - Capture: Capture from VHS/DV tapes (requires capture hardware)
  - Batch: Queue multiple videos for processing

================================================================================
  STEP 4: RESTORE YOUR FIRST VIDEO
================================================================================

RESTORATION TAB:

1. Click "Select Input File" - Choose your VHS capture or old video
2. Click "Select Output File" - Choose where to save the restored video
3. Configure settings:
   
   ESSENTIAL SETTINGS:
   - Field Order: "Auto-Detect" (recommended for VHS)
   - QTGMC Preset: "Slow" (good quality/speed balance)
   - Codec: "libx264 (H.264, CPU)" (universal compatibility)
   - CRF: 18 (lower = better quality, 18 is excellent)

   ADVANCED SETTINGS (optional):
   - Denoise: "Medium (Slow)" for noisy VHS tapes
   - Remove Artifacts: Enable for combing/chroma issues
   - AI Upscaling: Enable to upscale SD→HD (requires GPU)

4. Click "Start Processing"
5. Monitor progress bar and live log output

TYPICAL PROCESSING TIME: 
  - 1 hour VHS tape = 2-4 hours processing (depending on settings)
  - Faster preset = quicker processing, slightly lower quality
  - Slower preset = better quality, longer processing

================================================================================
  STEP 5: CAPTURE FROM TAPE (OPTIONAL)
================================================================================

CAPTURE TAB (requires capture hardware):

HARDWARE SETUP:
1. Connect your capture device:
   - Analog: USB capture card (Elgato, AVerMedia, Hauppauge)
   - DV/miniDV: FireWire (IEEE 1394) connection
2. Connect tape player to capture device:
   - VHS/Hi8: Use composite (RCA) or S-Video cables
   - DV/miniDV: Use FireWire cable from camera/deck
3. Power on player and insert tape

CAPTURE CONFIGURATION:
1. Click "Refresh Devices" to detect capture hardware
   - Video devices will populate in dropdown
   - Audio devices will populate separately

2. Select device type:
   - "Analog (VHS/Hi8/Video8/Betamax)" for analog sources
   - "DV/miniDV (FireWire)" for digital tape sources

3. Select devices:
   - Video Device: Choose your capture card
   - Audio Device: Usually same as video device
   - Video Input: Choose input source (Composite, S-Video, etc.)

4. Configure capture settings:
   RESOLUTION:
   - 720x480 (NTSC - US/Japan VHS)
   - 720x576 (PAL - Europe VHS)

   FRAME RATE:
   - 29.97 fps (NTSC)
   - 25 fps (PAL)

   CODEC (Lossless formats):
   - HuffYUV (fastest, recommended) - ~60-80 GB/hour
   - FFV1 (best compression) - ~40-60 GB/hour
   - Lagarith (Windows-compatible) - ~50-70 GB/hour
   - Copy (DV only - preserves DV stream) - ~13 GB/hour

5. **IMPORTANT** Choose output folder:
   - Click "Browse..." button
   - Select drive with sufficient free space
   - Recommended: External USB 3.0 drive
   - Setting is saved for future captures

6. Optional settings:
   - Duration: Set tape length in seconds (or leave blank for manual stop)
   - "Auto-restore after capture": Check to automatically process after capture

7. NEW IN v3.0 - ADVANCED CAPTURE FEATURES:

   CAPTURE PRESETS:
   - Use the "Capture Preset" dropdown to quickly apply common configurations
   - Built-in presets:
     * VHS NTSC/PAL (Composite)
     * S-VHS NTSC/PAL (S-Video)
     * DV NTSC/PAL (FireWire)
     * Component NTSC/PAL (YPbPr)
   - "Save as Preset" button saves your current settings as custom preset
   - Presets configure codec, resolution, framerate, and input sources

   LIVE PREVIEW:
   - Check "Enable Live Preview" to see video signal before/during capture
   - Opens separate window showing real-time video feed
   - Useful for verifying signal quality and framing
   - Disable to reduce CPU usage during capture
   - Warning: May impact capture performance on slower systems

   REAL-TIME MONITORING:
   - "Dropped Frames" counter shows capture quality
     * Green (0 frames): Perfect capture
     * Yellow (1-9 frames): Acceptable (minor glitches)
     * Red (10+ frames): Warning! Check disk speed or close other apps
   - "Disk Space" monitor shows available space on output drive
     * Updates during capture
     * Color-coded warnings: Red (<50GB), Orange (<100GB)
   - Both monitors update every second during active capture

CAPTURING:
1. Cue tape to starting position
2. Click "Start Capture" button
3. Press PLAY on tape player
4. Monitor capture:
   - Status shows active capture
   - File grows in real-time
   - Console shows capture progress

5. Stop capture:
   - Click "Stop Capture" when tape finishes
   - Or let it auto-stop if duration was set

AFTER CAPTURE:
1. Captured file saved to your chosen folder
   - Filename format: capture_YYYYMMDD_HHMMSS.avi
   - Immediately usable with any video software

2. If "Auto-restore" was enabled:
   - Dialog prompts to process captured file
   - Click "Yes" to restore immediately
   - Click "No" to process later

FILE LOCATIONS:
- Captures saved to YOUR chosen folder (any drive)
- Standard formats (AVI/MKV) playable in VLC, Premiere, Resolve
- Lossless quality preserves maximum detail for restoration

STORAGE PLANNING:
- SD capture (720x480, HuffYUV): ~60-80 GB per hour
- PAL capture (720x576, HuffYUV): ~70-90 GB per hour
- DV capture (copy mode): ~13 GB per hour
- Plan for 100+ GB free space for 2-hour tapes

VIDEO INPUT SOURCE SELECTION:
- Auto (Default): Let device choose (recommended for first-time use)
- Composite (RCA): Yellow RCA connector (standard VHS)
- S-Video (Y/C): Y/C connector (better quality for S-VHS)
- Component (YPbPr): Red/Green/Blue connectors (highest quality analog)
- HDMI/Digital: Digital input (professional cards only)

Note: If capture fails with a specific input, try "Auto (Default)"

TROUBLESHOOTING:
- No devices found: Check USB connections, reinstall drivers
- Capture fails: Ensure output folder is writable, check disk space
- Wrong input source: Use "Video Input" dropdown to switch
- Choppy video: Use faster drive (SSD recommended for HD)
- Audio sync issues: Verify audio device selected, check sample rate

For detailed capture guide, see CAPTURE_GUIDE.md

================================================================================
  TIPS & TRICKS
================================================================================

PERFORMANCE:
  - Close other applications during processing
  - QTGMC presets: Draft (fastest) → Placebo (best)
  - GPU denoising much faster than CPU if you have NVIDIA GPU

QUALITY:
  - Always use lossless capture (HuffYUV/FFV1) for best source
  - Test short clips first to find ideal settings
  - CRF 18 is excellent, 16 is near-lossless, 20 is very good

TROUBLESHOOTING:
  - If processing fails, check "last_error_log.txt" in app folder
  - Enable "Debug Logging" in settings for detailed output
  - Field order issues? Use Auto-Detect or try TFF/BFF manually

BATCH PROCESSING:
  - Add multiple files to queue via Batch tab
  - Queue persists across app restarts
  - Perfect for overnight processing of entire tape collection

================================================================================
  OUTPUT FORMATS
================================================================================

RECOMMENDED FORMATS:
  - MP4 (H.264): Universal playback, good compression
  - MP4 (H.265): Better compression, requires modern players
  - MOV (ProRes): Professional archival, large files
  - MKV (FFV1): Lossless archival, very large files

CODEC COMPARISON:
  - H.264: 1 hour VHS ≈ 2-4 GB (CRF 18)
  - H.265: 1 hour VHS ≈ 1-2 GB (CRF 18)
  - ProRes 422: 1 hour VHS ≈ 30-50 GB
  - FFV1: 1 hour VHS ≈ 50-100 GB (lossless)
