GETTING STARTED (2 STEPS):
--------------------------

1. INSTALL PREREQUISITES (One-Time, 5-10 minutes)
   - Double-click: FIRST_TIME_SETUP.bat
   - Wait for automatic installation to complete
   - RESTART your computer when prompted

2. LAUNCH APPLICATION
   - Double-click: Advanced_Tape_Restorer_v3.1.exe
   - Click "Select Input" to choose your video file
   - Choose restoration options (deinterlace, denoise, etc.)
   - Click "Start Processing"

That's it! Your restored video will be saved to the output folder.

===============================================================================
  WHAT GETS INSTALLED
===============================================================================

FIRST_TIME_SETUP.bat automatically downloads and installs:
  ✓ FFmpeg - Video encoding/decoding (~100 MB)
  ✓ VapourSynth - Video processing framework (~50 MB)
  ✓ VapourSynth Plugins - QTGMC, filters (~30 MB)

Total: ~180 MB download, ~300 MB installed
No Python required - everything is bundled in the .exe!

===============================================================================
  COMMON USE CASES
===============================================================================

1. DEINTERLACE VHS/MINIOV:
   - Load video
   - Enable "Deinterlace (QTGMC)"
   - Select preset: "Medium" (good balance)
   - Process

2. REMOVE NOISE FROM OLD FILM:
   - Load video
   - Enable "Denoise (BM3D)"
   - Set strength: 3-5 for heavy noise
   - Process

3. UPSCALE SD TO HD:
   - Load video
   - Enable "Manual Resize" (or "AI Upscale 2x" if GPU available)
   - Set target resolution: 1920x1080
   - Process

4. STABILIZE SHAKY FOOTAGE:
   - Load video
   - Enable "Stabilization"
   - Process

You can combine multiple filters! Try: Deinterlace + Denoise + Upscale

===============================================================================
  SUPPORTED INPUT FORMATS
===============================================================================

Video Files:
  • AVI (DV, HuffYUV, Lagarith, uncompressed)
  • MP4 (H.264, H.265, MPEG-4)
  • MKV (Matroska - all codecs)
  • MOV (QuickTime - ProRes, DV, H.264)
  • MPG/MPEG (MPEG-1, MPEG-2)
  • VOB (DVD Video)
  • TS/M2TS (Blu-ray, AVCHD)
  • WMV, FLV, WebM

Any resolution: 320×240 to 4K (3840×2160)
Any framerate: 23.976 to 60 fps

===============================================================================
  OUTPUT OPTIONS
===============================================================================

RECOMMENDED CODECS:
  • H.264 (CPU) - Universal playback, good quality
  • H.264 NVENC (GPU) - 5-10x faster on NVIDIA GPUs
  • H.265 (CPU) - Better compression for 4K
  • ProRes 422 - Professional editing
  • FFV1 - Lossless archival

QUALITY SETTINGS:
  • CRF 18 - Visually lossless (large files)
  • CRF 20-22 - High quality (recommended)
  • CRF 23-25 - Good quality (smaller files)

PRESET SPEED:
  • Slower - Best quality, slow encoding
  • Medium - Good balance (recommended)
  • Fast/VeryFast - Quick encoding, lower quality

===============================================================================
  TROUBLESHOOTING
===============================================================================

"vspipe not found" error:
  → Run FIRST_TIME_SETUP.bat to install VapourSynth
  → Make sure to RESTART computer after installation

"FFmpeg not found" error:
  → Run FIRST_TIME_SETUP.bat to install FFmpeg
  → Restart computer after installation

Processing is very slow:
  → Use NVENC encoding if you have NVIDIA GPU
  → Lower QTGMC preset (Faster, Fast)
  → Reduce output resolution
  → See README.txt for performance optimization tips

Out of memory errors:
  → Close other applications
  → Reduce "Num Threads" in settings
  → Process shorter segments
  → Use H.264 instead of ProRes
