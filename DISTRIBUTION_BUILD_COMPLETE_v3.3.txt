UNINSTALLATION WORKFLOW
========================

Complete Removal (Including Prerequisites):
  See: UNINSTALL_GUIDE.txt for detailed instructions

Quick summary:
  1. Delete application folder
  2. Uninstall VapourSynth (Windows Settings)
  3. Delete FFmpeg folder (C:\ffmpeg\)
  4. Uninstall PyTorch: pip uninstall torch torchvision torchaudio
  5. Delete config: %LOCALAPPDATA%\Advanced_Tape_Restorer
  6. Remove PATH entries (sysdm.cpl → Environment Variables)
  7. Restart computer
