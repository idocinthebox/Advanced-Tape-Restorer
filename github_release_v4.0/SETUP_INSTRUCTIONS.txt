========================================
Advanced Tape Restorer v4.0 - Setup
Community Edition (FREE - MIT Licensed)
========================================

QUICK START:

1. Install FFmpeg (required)
   Run: Install_FFmpeg.bat
   OR: winget install Gyan.FFmpeg

2. Install VapourSynth (required)
   Run: Install_VapourSynth.bat
   Download from: https://github.com/vapoursynth/vapoursynth/releases/tag/R73

3. Install VapourSynth plugins (required)
   Run: Install_VS_Plugins.bat
   (After installing VapourSynth)

4. Launch Application
   Run: Advanced_Tape_Restorer_v4.0.exe

========================================
TROUBLESHOOTING
========================================

Q: "FFmpeg not found" error
A: Restart terminal after installing FFmpeg, or add C:\ffmpeg\bin to PATH

Q: "VapourSynth not found" error
A: Run Install_VapourSynth.bat, extract to C:\VapourSynth, add to PATH

Q: "QTGMC is not defined" error
A: Run Install_VS_Plugins.bat to install required VapourSynth plugins

Q: Application won't start
A: Install Visual C++ Redistributable 2015-2022 from:
   https://aka.ms/vs/17/release/vc_redist.x64.exe

Q: Hardware capture devices not detected
A: Check Device Manager, install capture card drivers
