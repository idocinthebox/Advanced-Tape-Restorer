# Version and Product Matrix

## Current Public Split

| Offering | License/Commercial Model | Distribution | Notes |
| --- | --- | --- | --- |
| Community Edition (v4.0 branch identity) | MIT | This public repository | Free community codebase |
| ATR 6.0 Professional | Commercial | Separate product channel | Not included in this repository |
| LTX Generative Repair | Paid add-on | Separate add-on channel for ATR 6.0 Professional | Not included in Community Edition |

## Practical Guidance

- Use Community Edition if you want the open MIT branch.
- Use ATR 6.0 Professional if you require the separate commercial workflow.
- Add LTX Generative Repair only when that paid add-on is needed with ATR 6.0 Professional.
