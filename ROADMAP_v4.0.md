# Roadmap v4.0 (Community Edition)

## Purpose

This roadmap documents the Community Edition direction for the public MIT repository.

## Completed Foundation (Historical)

- Community restoration workflow for analog and DV sources
- Core deinterlacing and cleanup pipeline in the public codebase
- Community capture/export path maintained in the v4.0 line

## Community Maintenance Scope

The Community Edition roadmap is maintenance-first:

- Stability fixes
- Compatibility updates for core dependencies
- Documentation clarity for community users and contributors

## Product-Line Note

Commercial development continues outside this repository as ATR 6.0 Professional, with paid add-ons such as LTX Generative Repair managed separately.

## Out of Scope for This Roadmap

- Commercial pricing/packaging strategy
- Proprietary feature planning for paid editions
- Marketplace entitlement logic
